# 💳 Online Payment Fraud Detection

## 📌 Project Overview

This project detects fraudulent online payment transactions using Machine Learning.

The model is trained using Logistic Regression and deployed using Streamlit to provide real-time fraud prediction.

---

## 🚀 Features

- Fraud Detection using Machine Learning
- Logistic Regression Model
- Streamlit Web Application
- Data Preprocessing
- Fraud Prediction Interface

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Jupyter Notebook
